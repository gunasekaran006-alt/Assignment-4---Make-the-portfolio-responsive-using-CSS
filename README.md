# Responsive Personal Portfolio 🚀

Welcome to my portfolio! This is a clean, modern, and fully responsive website designed to showcase my journey as an **Aspiring Full Stack Developer**.

## ✨ Features
- **Fully Responsive:** Works perfectly on Mobile, Tablet, and Desktop using CSS Media Queries.
- **Modern UI:** Clean design using the 'Poppins' Google Font.
- **Project Showcase:** Section to display my latest web development work.
- **Contact Form:** A functional layout for users to get in touch.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, Flexbox for layouts, and Media Queries for responsiveness.
- **Google Fonts:** Integrated 'Poppins' for better typography.
